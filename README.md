# todoTasks- React
## What is TodoTask?

[TodoTask] React example  .

## Requirements

* Node
* Git


## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/RamikhouriRobots/react-todo-example.git
```

```bash
npm install
```

## Steps to run

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) and take a look around.
